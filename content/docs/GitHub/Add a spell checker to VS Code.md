# Add a spell checker to VS Code

## Install Code Spell Checker

1. In VS Code, select `View` ‚ùØ `Extensions`.
2. Search `spell checker`.
3. Install `Code Spell Checker` by Street Side Software.

## Find and fix spelling mistakes

Unknown words are underlined as you type. Find suggestions by selecting the word, then clicking the light bulb icon `üí°`.

Find all unknown words by selecting `View` ‚ùØ `Problems`.
